# Fallout 76 Request List

> A community-maintained database of every Atomic Shop item and bundle currently requestable through Bethesda Support.

---

## 🎯 Purpose

This project exists to make it easy for Fallout 76 players to browse everything that can currently be requested through Bethesda Support.

Instead of searching Reddit, YouTube, archived Atomic Shop posts, and the Fallout Wiki, this repository brings everything together in one place.

Each entry will eventually include:

- 📷 Preview images
- 📦 Bundle contents
- 🏷 Categories
- 🔗 Fallout Wiki links
- 💰 Original Atom price
- 📝 Notes
- ✅ Requestable status
